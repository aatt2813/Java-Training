This repository contains Java activities organized by modules. Each module has its own branch, keeping the repository clean and focused.

## Branches

- **master**  
  The `master` branch contains only this README file. It serves as the main entry point of the repository and keeps it clean.  

- **M1_Activity**  
  Contains all Java activities for **Module 1**. Work specific to Module 1 should be found and run in this branch.  

- **M2_Activity**  
  Contains all Java activities for **Module 2**. Work specific to Module 2 should be found and run in this branch.
